Request Fields

필드명 타입 필수값 양식 설명 제약조건

loginId

String

true

로그인 아이디

Must match the regular expression [a-zA-Z\d]{5,20} Must not be blank

password

String

true

비밀번호

Must match the regular expression (?=.[A-Za-z])(?=.\d)(?=.[$@!%?&])[A-Za-z\d$@!%*?&]{8,20} Must not be blank

name

String

true

이름

Must match the regular expression [a-zA-Z가-힣]{2,20} Must not be blank

nickname

String

true

닉네임

Must not be blank Size must be between 0 and 20 inclusive

email

String

true

이메일

Must match the regular expression [0-9a-zA-Z]([-\.]?[0-9a-zA-Z])@[0-9a-zA-Z]([-\.]?[0-9a-zA-Z])\.[a-zA-Z]{2,3} Must not be blank

birth

String

true

생년월일

Must be in the past Must not be null

gender

String

true

성별 코드

Must not be null

POST /api/v1/members/sign-up HTTP/1.1
Content-Type: application/json;charset=UTF-8
Accept: application/json
Content-Length: 201
Host: api.liveamonth.com

  "loginId" : "testID",
  "password" : "testPassword1!",
  "name" : "테스트",
  "nickname" : "테스트",
  "email" : "testEmail@gmail.com",
  "birth" : "2022-11-10",
